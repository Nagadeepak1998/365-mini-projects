Expanded a full-viewport cinematic streaming hero into a complete streaming homepage experience in React and Tailwind CSS.

What I focused on:
- real background video layering
- bottom-only blur with CSS mask
- reusable liquid-glass buttons
- staggered blur-fade animation
- responsive mobile navigation
- Movies, TV Series, Editor's Pick, Interviews, and User Reviews sections
- working section anchors from the navbar
- Lucide React icons

This is a small frontend portfolio piece, but it shows implementation discipline: translating a detailed UI prompt into a runnable, responsive React experience with real page depth that can live on GitHub and be reused in my portfolio.

Tech stack: React, Vite, Tailwind CSS, Lucide React, Inter font.
