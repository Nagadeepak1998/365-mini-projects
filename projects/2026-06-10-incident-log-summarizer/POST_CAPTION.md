Built a small production-style DevOps/AI project: Incident Log Summarizer.

It takes raw Kubernetes, CI, or application logs and converts them into a recruiter-readable incident report with severity, evidence lines, probable cause, and runbook-style next actions.

What I included:
- Python CLI package
- Deterministic incident signal detection
- Markdown and JSON output
- Pytest coverage
- Dockerfile
- GitHub Actions CI
- Real sample logs for demo

This project connects my production support, DevOps, platform engineering, and AI automation interests without relying on fake API calls or hardcoded secrets.
