# Seconds to Human (Shell CLI)

Tiny shell utility that converts a non-negative integer number of seconds into a readable `days hours minutes seconds` format.

## Usage

```bash
./seconds_to_human.sh <seconds>
```

## Example

```bash
./seconds_to_human.sh 98765
# 1d 03h 26m 05s
```

## Why this mini-project?

Useful for quick log/runtime interpretation in terminal workflows.
